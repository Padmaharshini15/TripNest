# TripNest Frontend

TripNest Frontend is the client-side application of the TripNest travel booking platform. It provides an interactive and responsive user interface for browsing destinations, viewing details, making bookings, and managing user accounts.

## Features

- Browse travel destinations
- Search and filter destinations
- Destination details page
- User authentication (JWT)
- Booking system
- Responsive UI
- Admin dashboard support
- REST API integration with Spring Boot backend

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Vite
- REST API
- JWT Authentication

## Installation

1. Clone the repository

```
git clone <repository-url>
```

2. Navigate to the project

```
cd tripnest-frontend
```

3. Install dependencies

```
npm install
```

4. Start the development server

```
npm run dev
```

## Backend

The frontend communicates with the TripNest Spring Boot Backend through REST APIs.

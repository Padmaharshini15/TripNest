# 🌍 TripNest – Tourism Management System

TripNest is a full-stack Tourism Management System developed using Java Spring Boot for the backend and HTML, CSS, JavaScript (Vite) for the frontend.

The application allows users to explore tourist destinations, manage bookings, maintain wishlists, submit reviews, and provides an admin panel for destination and booking management.

---

## 📌 Features

### 👤 User Features
- User Registration & Login (JWT Authentication)
- View Tourist Destinations
- Search & Filter Destinations
- Add & Remove Wishlist
- Book Tourist Packages
- View & Cancel Bookings
- Add Reviews
- Manage User Profile

### 👨‍💼 Admin Features
- Secure Admin Login
- Add Tourist Spots
- Update Tourist Spots
- Delete Tourist Spots
- View All Bookings
- Update Booking Status
- Dashboard Analytics

# 🚀 Getting Started

## Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/tripnest.git
```

---

## Backend Setup

```bash
cd tripnest-backend
```

Configure MySQL in

```
application.properties
```

Run

```bash
mvn spring-boot:run
```

Backend runs at

```
http://localhost:8080
```

---

## Frontend Setup

```bash
cd tripnest-frontend
npm install
npm run dev
```

Frontend runs at

```
http://localhost:3000
```
